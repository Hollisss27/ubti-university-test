# UBTI 大学气质测试

一个面向高考季传播场景的互动测试网站。用户完成 22 道轻量题目后，会得到一个对应的“大学气质”角色、稀有度、解读、气质雷达图和可分享海报。

项目灵感来自 MBTI/SBTI 类互联网测试的传播形式，但内容围绕“高考结束后，你适合上什么大学气质”重新策划，包括题库、结果体系、角色形象和分享页。

## 在线体验

https://ubti.eu.cc/

## 你会看到什么

- 22 道轻量题目，围绕高考结束、大学开学、校园日常和精神状态展开。
- 25 个大学气质角色，每个角色都有自己的名称、形象、稀有度和解读。
- 测试结果包含气质雷达图、契合度、校园出没地和可分享海报。
- 页面适配手机端，适合刷到后快速测一下、笑一下、顺手分享给朋友。

## 适合谁玩

- 刚高考结束，正在估分、放空、等成绩的人。
- 已经离开考场很多年，但每到六月还是会想起那个夏天的人。
- 想看看自己更像哪种“大学气质”的人。
- 想找一个轻松一点的话题发给朋友玩的人。

## 项目特点

- 不采集用户答题数据，所有计算都在浏览器本地完成。
- 题库、结果体系、角色文案和分享页围绕高考季传播场景设计。
- 整体为纯静态网页，方便部署和访问。

## 说明

本项目为娱乐向互动测试，结果不代表真实择校建议。

角色图片、题目文案与结果解读均为项目定制内容，仅用于本项目展示与体验。如需二次使用或改编，请保留项目说明并注明来源。
